# YourPy

## Package
Python 
~~~python
from dir.sub1.sub2 import Name
~~~

YourPy
~~~python
from dir.sub1.sub2 import Name
# and
from package name import Name
~~~

## Block
Python
~~~python
for x in [1,2,3]:
  print(x)
~~~

YourPy
~~~python
for x in [1,2,3]:
  print(x)
# and
for x in [1,2,3]{
  print(x)
}
~~~
